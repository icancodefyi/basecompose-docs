# Development Setup

## Prerequisites
- Node.js 18+
- pnpm 8+
- MongoDB 7+

## Setup Steps
1. Clone repo
2. Install dependencies: `pnpm install`
3. Create `.env.local` (see example)
4. Start MongoDB
5. Run: `pnpm dev`
