# Contribution Guide

We welcome all contributions!

## How to Contribute
- Report bugs (GitHub Issues)
- Suggest features (GitHub Discussions)
- Submit code (see CONTRIBUTING.md)
- Improve documentation

## Development Workflow
1. Fork and clone repo
2. Create a feature branch
3. Make changes, add tests/docs
4. Commit with conventional message
5. Push and create PR

## Adding Technologies
- Edit `packages/types/stack-config.ts`
- Update types in `packages/types/blueprint.ts`
- Test in dev server

See [CONTRIBUTING.md](https://github.com/icancodefyi/basecompose/blob/main/CONTRIBUTING.md)
