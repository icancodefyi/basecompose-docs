# FAQ & Troubleshooting

## Common Issues
- MongoDB connection errors
- Missing environment variables
- Port conflicts
- Hot reload not working

## Solutions
- Ensure MongoDB is running
- Check `.env.local` setup
- Kill processes on port 3000
- Use dev compose for hot reload

See [DEVELOPMENT_SETUP.md](https://github.com/icancodefyi/basecompose/blob/main/DEVELOPMENT_SETUP.md)
