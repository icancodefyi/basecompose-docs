# Environment Variables

Generated projects include `.env.example` covering:
- MongoDB: `MONGODB_URI`, `MONGODB_ROOT_USERNAME`, `MONGODB_ROOT_PASSWORD`
- Auth.js: `GITHUB_ID`, `GITHUB_SECRET`, `NEXTAUTH_URL`, `NEXTAUTH_SECRET`

See [DEVELOPMENT_SETUP.md](https://github.com/icancodefyi/basecompose/blob/main/DEVELOPMENT_SETUP.md) for setup.
